# DeckOS Docs

This guide is written for people using DeckOS to manage a Linux homelab server. It focuses on the tasks you do from the DeckOS web interface and the basic host operations needed to keep the service healthy. If you are brand new to DeckOS, start with installation and then work through the getting-started guide before moving into the deeper sections.

## What DeckOS Helps You Do

- Watch system health from a browser
- Deploy Docker Compose apps from templates or custom YAML
- Start, stop, restart, and update managed app stacks
- View logs and edit compose files without leaving the UI
- Browse host files, upload and download files, and edit text content
- Apply DeckOS updates and roll back to the previous release if needed

## Before You Begin

- DeckOS is designed for trusted environments such as a home network or private lab
- The provided installer supports Ubuntu `24.04`, `25.10`, and `26.04`
- DeckOS manages Docker on the host and therefore has powerful access to the machine

If you are installing DeckOS for the first time, begin with [Install DeckOS](install.md).
